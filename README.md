# （開発中）claude-code-client

## 概要
Claude CodeをWebブラウザから操作可能にするアプリケーションサーバ。

### 本アプリケーションがもたらす恩恵
いつでもどこでもClaudeが実施している開発状況を確認し、支持が出せるようになる。

### 操作イメージ
1. 外出先で、スマートフォンからブラウザ経由で本アプリケーションにアクセスする。
2. ユーザの認証をする。
3. 前回実行までのセッションを保持したClaudeと、チャット形式でやり取りができる。
4. ブラウザ上でTerminalの操作ができる。

### 機能要件
1. 本アプリケーションがインストールされた環境を、Claudeの作業スペースとする。
2. ブラウザ上で、Terminalの実行ができる。
3. ブラウザ上で、チャット形式でClaudeに作業指示が出せる。
4. ブラウザ上で、Claudeからの権限承認を確認できる。
5. ブラウザ上で、Claudeの作業状況を確認できる。
6. ブラウザ上で、作成中のアプリのデプロイができる。
7. ブラウザ上で、複数のClaudeセッションを管理できる。
8. プッシュ通知で、Claudeからの作業確認と、完了報告を受け取れる。
9. Claudeからの通知は、Webhook登録が可能で、LINEやSlack、discodeなどと連携できる。
